# 简介

`Puock` 专属插件,用于生成海报,打赏,分享二维码,emoji表情,前端ajax登录等功能
不使用插件则以上功能不能使用.

下载本项目到 `/usr/plugins/` 目录下并解压

更改插件目录名为 `Puock`
